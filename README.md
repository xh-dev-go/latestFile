# Usage
```shell
Please input directory pathUsage of latestFile.exe:
  -delay int
        the time delay for check directory stat (default -1)
  -directory string
        the path to detect
  -kafka-host string
        kafka host
  -kafka-topic string
        kafka topic
  -version
        check the version fo application
```

# Demo
```shell
./latestFile --directory {directory} --kafka-host {kafka-host} --kafka-topic {kafka-topic}
```
